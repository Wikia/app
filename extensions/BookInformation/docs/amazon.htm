Amazon Web Services for BookInformation

The Amazon book information driver allows the BookInformation extension to use Amazon Web Services as a provider for ISBN lookup data.

Prerequisites

You will need an Amazon Web Services account, and you will need to know your Access Key ID.

Configuration

$wgBookInformationProvider = 'Amazon';
$wgBookInformationService = array(
	'accesskeyid' => {{ACCESSKEYID}},
);

where {{ACCESSKEYID}} is your Access Key ID. This can be obtained through the Amazon Web Services developer centre.

Available Information

  • Title
  • Author
  • Publisher

Other

  • The data provider will be listed as "Amazon Web Services"
  • A link to purchase the book through Amazon.com will be displayed
