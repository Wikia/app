Remove @import rules and re-add

Quirkiness

IE will trigger some kind of soft crash (CPU hits 100% and process must be killed with force) when adding @import rules to STYLE elements.

Find and remove any @import rule and make a LINK

domobject_ie

IE OBJECT element

Quirkiness

Internet Explorer OBJECT elements have especially strange behavior.  Instead of being able to call <appendChild>() to add PARAM elements, this must be done using innerHTML on an anonymous DIV
